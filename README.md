## dotfiles

> install

## pyenv
### install

> install

    $ brew install pyenv
    $ brew install pyenv-virtualenv

> update

    $ pyenv update

> python install

    $ pyenv install 2.7.6

> 切り替え

    $ pyenv local 2.7.6

* pyenv-virtualenv

> 仮想環境作成

    $ pyenv virtualenv 2.7.6 kemono-2.7.6

> 仮想環境選択

    $ pyenv local kemono-2.7.6
